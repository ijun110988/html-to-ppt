Workflow Status & Monitoring

Sistem tracking order visual untuk prioritas kerja tim fulfillment

Pending

Order baru masuk via Google Form. Menunggu pembayaran dari customer.

Paid

Pembayaran dikonfirmasi. Admin update status ini untuk trigger proses packing.

Shipped

Barang dikirim & resi diinput. Sistem otomatis kirim notifikasi WA.

Done

Paket diterima customer. Order selesai dan masuk rekap omzet bulanan.

Gunakan "Filter View" di Google Sheets untuk sort order per status
Color Coding memudahkan monitoring ribuan data

Dengan status workflow, Anda dan tim bisa langsung tahu prioritas. Filter status PAID untuk lihat order yang harus diproses hari ini. Filter SHIPPED untuk monitor yang perlu follow up. Sistem ini membuat monitoring jadi sangat mudah.
